# DT solution - Next.js Starter

This is a minimal Next.js scaffold implementing the provided layout (hero, services, testimonials, blog, FAQ, footer with contact form) for DT solution.

Quick start:

1. Install dependencies

```bash
npm install
```

2. Run the dev server

```bash
npm run dev
```

Open http://localhost:3000

Files created:
- `pages/_app.js`, `pages/index.js`
- `components/*` (Header, Hero, Services, Testimonials, Blog, FAQ, Footer, ContactForm)
- `styles/globals.css`

You can now refine content, add images under `public/`, and wire the contact form to an API route.
